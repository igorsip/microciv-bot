# 🏛 MicroCiv - Telegram Strategy Game

Стратегическая игра для Telegram, вдохновленная Civilization. Управляй своей цивилизацией, строй здания, исследуй технологии!

## 🎮 Возможности (Текущая версия v1.0)

✅ **Реализовано:**
- 🤖 Telegram бот с командами
- 🎨 Красивый игровой интерфейс (Mini App)
- 💰 Система ресурсов (еда, производство, наука)
- 🏗️ 5 типов зданий (ферма, казарма, библиотека, рынок, мастерская)
- ⏱️ Генерация ресурсов в реальном времени
- 💾 Сохранение прогресса в базе данных
- 📊 Статистика игрока

🚧 **В разработке** (следующие этапы):
- 🔬 Дерево технологий
- ⚔️ Военная система
- 🗺️ Карта мира
- 💎 Монетизация через Telegram Stars
- 🎯 Достижения и квесты

## 📋 ИНСТРУКЦИЯ ПО УСТАНОВКЕ

### Шаг 1: Установка Node.js

**Windows:**
1. Скачай установщик с [nodejs.org](https://nodejs.org/)
2. Выбери версию **LTS** (рекомендуемая)
3. Запусти установщик и следуй инструкциям
4. Перезагрузи компьютер

**Mac:**
```bash
# Через Homebrew
brew install node
```

**Linux (Ubuntu/Debian):**
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**Проверка установки:**
```bash
node --version
npm --version
```

Должно показать версии (например: v20.10.0 и 10.2.0)

---

### Шаг 2: Создание Telegram бота

1. **Открой Telegram** и найди [@BotFather](https://t.me/botfather)

2. **Отправь команду:**
   ```
   /newbot
   ```

3. **Придумай имя бота:**
   ```
   MicroCiv Game Bot
   ```

4. **Придумай username** (должен заканчиваться на 'bot'):
   ```
   microciv_game_bot
   ```

5. **Скопируй токен** (выглядит примерно так):
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
   ```

6. **Настрой бота** (отправь BotFather):
   ```
   /mybots → выбери своего бота → Edit Bot → Edit Description
   ```
   Добавь описание:
   ```
   🏛 Стратегическая игра! Управляй цивилизацией, строй города, развивай технологии! 🎮
   ```

---

### Шаг 3: Установка проекта

1. **Открой терминал** (командную строку):
   - Windows: `Win + R` → введи `cmd` → Enter
   - Mac: `Cmd + Space` → введи `Terminal` → Enter
   - Linux: `Ctrl + Alt + T`

2. **Перейди в папку с проектом:**
   ```bash
   cd путь/к/папке/microciv-bot
   ```
   
   Пример для Windows:
   ```bash
   cd C:\Users\YourName\Desktop\microciv-bot
   ```

3. **Установи зависимости:**
   ```bash
   npm install
   ```
   
   Подожди 1-2 минуты, пока установятся все библиотеки.

---

### Шаг 4: Настройка конфигурации

1. **Создай файл `.env`** (скопируй `.env.example`):
   
   **Windows (командная строка):**
   ```bash
   copy .env.example .env
   ```
   
   **Mac/Linux:**
   ```bash
   cp .env.example .env
   ```

2. **Открой файл `.env`** в любом текстовом редакторе (Блокнот, VS Code, и т.д.)

3. **Вставь свой токен бота:**
   ```env
   BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
   PORT=3000
   SERVER_URL=http://localhost:3000
   DEV_MODE=true
   ```

4. **Сохрани файл**

---

### Шаг 5: Запуск бота (ЛОКАЛЬНО)

1. **В терминале выполни:**
   ```bash
   npm start
   ```

2. **Ты должен увидеть:**
   ```
   ╔═══════════════════════════════════════╗
   ║     🏛 MicroCiv Server Started 🏛    ║
   ╚═══════════════════════════════════════╝
   
   🌐 Server URL: http://localhost:3000
   📡 Port: 3000
   🤖 Bot: Starting...
   
   ✅ База данных инициализирована
   ✅ Telegram бот запущен
   
   ✅ Всё готово! Открой Telegram и напиши своему боту /start
   ```

3. **Открой Telegram** → найди своего бота → напиши `/start`

⚠️ **ВАЖНО:** Локально Mini App не будет работать (нужен публичный URL). Для тестирования используй `/stats` в боте.

---

### Шаг 6: Деплой в интернет (для работы Mini App)

Чтобы Mini App работал, нужен публичный URL. Есть несколько способов:

#### Вариант А: Ngrok (для быстрого тестирования)

1. **Установи ngrok:**
   - Скачай с [ngrok.com](https://ngrok.com/)
   - Зарегистрируйся и получи authtoken

2. **Запусти ngrok:**
   ```bash
   ngrok http 3000
   ```

3. **Скопируй URL** (например: `https://abc123.ngrok.io`)

4. **Обнови `.env`:**
   ```env
   SERVER_URL=https://abc123.ngrok.io
   ```

5. **Перезапусти бота** (`Ctrl+C` → `npm start`)

6. **Теперь Mini App будет работать!**

#### Вариант Б: Бесплатный хостинг Render.com

1. **Зарегистрируйся на [render.com](https://render.com/)**

2. **Создай новый Web Service:**
   - Подключи GitHub репозиторий
   - Build Command: `npm install`
   - Start Command: `npm start`

3. **Добавь Environment Variables:**
   - `BOT_TOKEN` = твой токен
   - `PORT` = 3000
   - `SERVER_URL` = https://твой-сервис.onrender.com

4. **Deploy!**

5. **Обнови webhook бота** (отправь через браузер):
   ```
   https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://твой-сервис.onrender.com
   ```

## 🐛 Возможные проблемы и решения

### Проблема: "Cannot find module 'node-telegram-bot-api'"
**Решение:**
```bash
npm install
```

### Проблема: "Error: Bad Request: wrong persistent menu button URL"
**Решение:** Проверь, что `SERVER_URL` в `.env` правильный и доступен.

### Проблема: Бот не отвечает
**Решение:**
1. Проверь, что токен правильный
2. Проверь, запущен ли сервер
3. Посмотри логи в терминале

### Проблема: Mini App не открывается
**Решение:**
1. Убедись, что используешь публичный URL (ngrok или Render)
2. Проверь, что `SERVER_URL` в `.env` указан правильно
3. Убедись, что сервер запущен
